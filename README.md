# md2html

`md2html` is a tiny Go program that reads Markdown from `stdin`, converts it to HTML, and writes to `stdout`.

## Install

```sh
go install github.com/jon-ski/md2html@latest
````

## Usage

```sh
cat file.md | md2html > file.html
```
